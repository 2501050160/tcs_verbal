# IntelliTest Verbal Ability Platform

IntelliTest is an automated, high-fidelity verbal comprehension evaluation and benchmarking simulator customized specifically for the **TCS NQT Verbal Ability** curriculum. 

## 🛡️ Administrative Security Portal

To secure the calibration controls and candidate grading matrices from unauthorized tampering, a master security lock has been applied.

- **System Admin Passcode**: `AdminVerbal2026`

Enter this security key on the **System Admin** tab to decrypt administrative ledger records, configure countdown duration parameters, delete records, or reset cohort benchmarks.

## 🚀 Key Architectural Features

- **Blanks Completion Engine**: A 20-blank random/sequential testing flow with persistent client-side timers.
- **Passage Recall Tracker**: Deep-focused memorization cards which track candidate recall using high-performance evaluation algorithms.
- **AI-Powered Email Grading**: Automatic text structure, keyword correlation, and tone auditing using high-contrast evaluator telemetry models.
- **Analytics & Credentials Display**: Peer comparisons, verifiable electronic performance diplomas, and detailed performance insights.
- **Developer & Design Focus**: Custom obsidian twilight theme utilizing Space Grotesk display headings paired with JetBrains Mono.
